# ☕ جرد الكافيه اليومي

تطبيق ويب كامل لإدارة جرد الطلبات اليومية لكافيه، يعمل بالكامل من المتصفح بدون أي Backend أو خادم، ويعتمد على **HTML + CSS + JavaScript** فقط (بدون أي Framework)، مع حفظ البيانات في `LocalStorage`.

## 📁 محتويات المشروع

```
index.html         الصفحة الرئيسية وهيكل الواجهة
style.css           التصميم الكامل (Modern UI - Coffee Brown)
script.js           كل منطق التطبيق (الحالة، الحفظ، التقارير...)
manifest.json        ملف PWA لدعم التثبيت على الشاشة الرئيسية
service-worker.js     دعم العمل بدون إنترنت (Offline)
README.md            هذا الملف
```

## 🚀 طريقة التشغيل

1. حمّل جميع الملفات في مجلد واحد بنفس الأسماء.
2. افتح ملف `index.html` مباشرة في أي متصفح حديث (Chrome, Edge, Safari, Firefox).
3. لأفضل تجربة (خصوصًا خاصية العمل بدون إنترنت عبر Service Worker)، يفضل تشغيل المشروع من خلال سيرفر محلي بسيط، مثال:

```bash
# باستخدام بايثون
python3 -m http.server 8080

# ثم افتح المتصفح على
http://localhost:8080
```

> ملاحظة: عند فتح الملف مباشرة بصيغة `file://` سيعمل التطبيق بشكل كامل (بما أنه لا يعتمد على أي خادم)، لكن تسجيل الـ Service Worker قد لا يعمل في بعض المتصفحات إلا عبر `http(s)://`. هذا لا يؤثر على عمل التطبيق نفسه لأن كل البيانات محفوظة أصلاً في `LocalStorage`.

## 🧭 كيفية الاستخدام

### 1. الصفحة الرئيسية
- تعرض أعلى الصفحة: تاريخ اليوم، اسم اليوم، إجمالي الطلبات، وعدد المنتجات المختلفة المباعة.
- كل منتج يظهر داخل Card فيه اسم المنتج، وزر `+`، والعدد الحالي، وزر `-`.
- الضغط على `+` يزيد العدد بواحد، والضغط على `-` ينقصه بواحد بدون النزول أقل من صفر.
- يظهر زر "تصفير" صغير أسفل أي منتج تم طلبه، لتصفير هذا المنتج فقط.

### 2. البحث
- استخدم مربع البحث أعلى الصفحة للعثور على أي منتج بكتابة اسمه مباشرة.

### 3. إنهاء اليوم
- زر "إنهاء اليوم" أسفل الصفحة يقوم بـ:
  - حفظ تقرير كامل لليوم الحالي (كل منتج وعدد مرات طلبه، الإجمالي، وأكثر منتج مبيعًا) داخل سجل الأيام.
  - إعادة تصفير كل الطلبات تلقائيًا وبدء يوم جديد فارغ.

### 4. سجل الأيام
- من أيقونة السجل أعلى الصفحة يمكنك:
  - استعراض كل الأيام السابقة (اسم اليوم + التاريخ + إجمالي الطلبات).
  - الضغط على أي يوم لعرض تقريره الكامل بالتفصيل.
  - حذف تقرير يوم معين، أو حذف جميع التقارير دفعة واحدة.

### 5. الإحصائيات
- من أيقونة الإحصائيات أعلى الصفحة يمكنك مشاهدة:
  - إجمالي طلبات اليوم الحالي.
  - عدد المنتجات المباعة اليوم.
  - أكثر منتج مبيعًا اليوم.
  - إجمالي عدد الأيام المسجلة في السجل.

### 6. أزرار إضافية
- **تصفير منتج واحد**: زر "تصفير" أسفل كل منتج تم طلبه.
- **تصفير جميع المنتجات**: زر "تصفير الكل" أسفل الصفحة (لا يحذف اليوم، فقط يعيد كل الأعداد لصفر).
- **حذف تقرير يوم معين**: من داخل تقرير اليوم المفتوح في نافذة السجل.
- **حذف جميع التقارير**: من داخل نافذة سجل الأيام.

## 💾 الحفظ التلقائي

كل تغيير (زيادة/نقصان/تصفير/إنهاء يوم) يُحفظ فورًا داخل `LocalStorage` بالمتصفح. عند إغلاق الموقع وإعادة فتحه، يكمل التطبيق تلقائيًا من آخر حالة محفوظة دون فقد أي بيانات.

## 🎨 التصميم

- الهوية اللونية: Coffee Brown (بني القهوة) مع خلفية فاتحة (Cream).
- Cards بحواف دائرية كبيرة (Border Radius) وظلال خفيفة (Soft Shadow).
- حركات (Animations) بسيطة عند الضغط على الأزرار لإحساس تفاعلي سلس.
- أيقونات لكل قسم (☕ 🥤 🧇) بالإضافة لأيقونات SVG بسيطة للأزرار الرئيسية.
- تصميم متجاوب بالكامل (Responsive) يعمل بسلاسة على الموبايل والتابلت والكمبيوتر.

## 🔒 الخصوصية

كل البيانات تبقى محليًا داخل متصفح الجهاز فقط (`LocalStorage`) ولا يتم إرسال أي بيانات لأي خادم خارجي، لأن التطبيق لا يحتوي على أي Backend.

## ⚠️ ملاحظة هامة

مسح بيانات المتصفح (Clear Browsing Data / Cache) الخاصة بهذا الموقع سيؤدي لحذف كل البيانات المحفوظة (اليوم الحالي وسجل الأيام السابقة)، لذا يُنصح بعدم مسح بيانات الموقع إذا كنت ترغب بالاحتفاظ بالتقارير.
